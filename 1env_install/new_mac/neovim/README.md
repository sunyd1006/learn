笔记
*nvim/lua/plugin 是编辑生成的代码，一般是不用动的
* 有些图标不显示，需要配置xxx



:PackerUpdate : 当需要更新现有插件到最新版本时使用。
:PackerSync : 在修改了插件配置（如新增或移除插件）后，用于同步插件状态。
:PackerCompile : 在修改了插件配置文件后，重新生成优化后的加载配置以提升性能。
